# Russellbrenner Happy

## How do I install these formulae?

`brew install russellbrenner/happy/<formula>`

Or `brew tap russellbrenner/happy` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "russellbrenner/happy"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
